# Hotelistan: Hotel Management System 
 
Hotelistan is a web-based service designed specifically for boutique hotels and hostels to streamline administrative tasks. Our goal is to replace disorganized paper-based or complex legacy systems with a user-friendly interface that requires minimal computer literacy from staff. 
 
## Features 
- **Real-time Room Tracking:** Monitor room conditions (e.g., available, occupied, dirty) across the system. 
- **Observer Pattern Integration:** Automated status updates (e.g., "Urgent Cleaning" requests) propagate instantly to relevant staff members. 
- **Role-Based Views:** Specialized interfaces for Receptionists, Housekeeping, and Guests, filtering out unnecessary menu items. 
- **Emergency Management:** Built-in alerts for urgent housekeeping or security needs. 
 
## Tech Stack 
- **Languages:** TypeScript 
- **Architecture:** Layered Architecture / OOD Analysis & Design (OOAD) 
- **Design Patterns:** Observer Pattern 
- **Modelling:** UML (Class, Sequence, and Use Case Diagrams) 
- **Management:** Scrum Methodology 
 
## Local Setup 
1. Clone the repository: `git clone <url>` 
2. Install dependencies: `npm install` 
3. Start the development server: `npm run dev`
